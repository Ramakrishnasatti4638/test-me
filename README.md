# Todo App

A simple, clean todo application built with vanilla HTML, CSS, and JavaScript — no dependencies required.

## Features

- **Add** tasks with the input field
- **Complete** tasks by clicking the circle checkbox
- **Delete** tasks with the ✕ button (appears on hover)
- **Edit** tasks by double-clicking the task text
- **Filter** tasks: All / Active / Completed
- **Clear completed** tasks in one click
- **Persistent** — todos are saved to `localStorage`

## Usage

Open `index.html` in any modern browser. No build step or server required.
